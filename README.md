# Scannator 
## Web Application Scanner Tool 

* Directory Scan
* Subdomain Scan

## Installation

`go get -u  github.com/erennuygun/scannator`

or

`git clone http://github.com/erennuygun/scannator.git`

`cd scannator`

`go build scannator.go`

`./scannator -h`
